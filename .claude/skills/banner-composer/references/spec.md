# Banner Composer — 规范说明

本规范中的 **安全区** 为**共享规范**：与 banner 相关的其他 Skill（如裁切、扩图、背景生成等）在实现时应引用并遵守，以便文案与画面主体均落在安全区内，避免被裁或构图失衡。后续新建相关 Skill 时请查阅本节及下方「安全区」定义。

**文字与坐标约定**：所有主标题、副标题、按钮等文字的位置与字号均以本规范及实现侧 `LAYOUT_BY_CANVAS` 中的**坐标像素与字号**为准，合成时严格按配置绘制。新增或修改某尺寸规范时，须在本文档与 `compose_banner.py` 的 `LAYOUT_BY_CANVAS` 中同时写明主标题 (x, y)、字号，副标题 (x, y)、字号；有按钮时写明框体与文字坐标、字号。

## 5. 安全区（共享规范）

- **安全区按画布尺寸规定为固定数值，不做比例换算**；更换画布尺寸后使用该尺寸对应的另一组固定值。
- **画布 W=1976、H=464**（规范1）安全区（固定）：
  - **x = 752～1457**
  - **y = 0～464**
- **画布 W=3320、H=500**（商店专题长图 3320\*460）安全区（固定）：
  - **x = 1470～2464**
  - **y = 0～500**
  - 顶部 **y = 0～40**：**纯白色背景**；在 **x = 1470～2464**、**y = 40～500** 区域内根据画面内容在纯白背景上可有画面内容填充；合成阶段顶条不二次盖白（见 `spec.py` / `prepare_background` A5b）。
- **画布 W=500、H=280**（专题封面）安全区：x=300～500，y=160～280。
- **画布 W=304、H=216**（田字格 304×216）安全区（固定）：x=0～303，y=0～120。
- **画布 W=1740、H=220**（专题头图）：安全区 x=510～1312，y=76～172；只做画面裁切对齐，无文字填充。
- **画布 W=984、H=442**（商店移动端banner 984×442）安全区（固定）：**x = 480～964**，**y = 30～382**。
- **画布 W=650、H=275**（商店移动端专题封面 650×275）安全区（固定）：**x = 300～630**，**y = 30～215**。
- **画布 W=720、H=220**（商店移动端专题头图 720×220）安全区（固定）：**x = 330～700**，**y = 30～160**。
- **画布 W=1196、H=672**（游戏中心小卡片 1196×672）安全区（固定）：**x = 50～1196**，**y = 95～672**；无文字填充，无遮罩；logo 填充区见 §5.7。
- 其他画布尺寸：各自对应一组固定数值（见实现侧按 (W,H) 或预设的配置）；未单独规定时由实现约定回退方式（如居中裁切）。
- **用途**  
  - **banner-composer**：主标题、副标题等文案须落在此区域内。  
  - **其他 Skill（裁切、扩图、背景生成等）**：裁切或扩图时，应将**核心画面/主体**控制在安全区内，避免核心内容落在安全区外导致被裁或与文案区域脱节。
- **引用**: 后续与 banner 相关的 Skill 在实现裁切、扩图、构图时，应引用本安全区规定并执行，无需另行推导。

## 5.5 图例区（legend zone）（共享规范）

- **图例区按画布尺寸规定为固定数值**，用于预留放置 logo、图例或角标等；与安全区并列，实现侧按 (W,H) 配置。
- **画布 W=1976、H=464**（规范1）图例区（固定）：**x = 1257～1457**，**y = 364～464**（右下 200×100px）。
- **画布 W=3320、H=500**（专题长图 3320×460）图例区（固定）：**x = 3100～3320**，**y = 420～500**（右下 220×80px）。
- **画布 W=500、H=280**（专题封面）图例区：**x = 400～500**，**y = 230～280**（右下 100×50px）。
- **画布 W=304、H=216**（田字格 304×216）图例区（固定）：**x = 204～304**，**y = 166～216**（右下 100×50px）。
- **画布 W=1740、H=220**（专题头图）图例区（固定）：**x = 1540～1740**，**y = 170～220**（右下 200×50px）。
- 其他画布尺寸：由实现侧在 `LAYOUT_BY_CANVAS` / `LEGEND_ZONE_BY_CANVAS` 中配置；未配置时无图例区。

## 5.6 联合 logo（Lengion zone 游戏中心小卡片联合 logo 50）（共享规范）

- **命名**：Lengion zone 游戏中心小卡片联合 logo 50。
- **布局（从左到右）**：
  - **图1（可替换 logo）**：对应用户上传的 logo（即「上传的图2 / 之前约定的上传 logo」）；合成时作为 `--logo1` 或 `assets/logo1.png` 传入；按整体高度等比缩放，宽度随比例。
  - **间距**：20px。
  - **中间 X 按钮**：22×22 px，无背景（透明）；仅绘制 X 线条，颜色 **#FFFFFF**；X 为两条对角线、交点位于中心，四臂等长，上下左右对称；在整体高度内垂直居中。
  - **间距**：20px。
  - **图2（固定 logo）**：固定为 LEGION ZONE 品牌 logo，不随上传变化；对应 `--logo2` 或 `assets/logo2.png`；按整体高度等比缩放，宽度随比例。
- **整体高度**：50px；图1、X 按钮、图2 均按 50px 高度垂直居中对齐。
- **实现与输出**：脚本 `scripts/combine_joint_logo.py`；输入为图1、图2 路径，输出默认 `output/joint_logo.png`；可通过 `--height` 覆盖整体高度（默认 50）。
- **资源约定**：图1、图2 可放置于 `assets/logo1.png`、`assets/logo2.png`，或通过参数 `--logo1`、`--logo2` 指定。

## 5.7 游戏中心小卡片 1196×672（共享规范）

- **命名**：游戏中心小卡片 1196×672。
- **画布尺寸**：1196×672 px。
- **画面安全区**：**x = 50～1196**，**y = 95～672**。
- **文字与遮罩**：没有文字填充，没有遮罩。
- **Logo 填充**：
  - 在 **x=40、y=50** 有 **500×55 px** 的 logo 填充区，使用「**Lengion zone 游戏中心小卡片联合 logo 50**」（规范见 §5.6）。
  - **对齐与缩放**：logo 对齐左上角，根据最大宽度做等比缩放。
  - **格式与处理**：logo 为透明背景的 PNG 格式。需要判断是否为透明背景 PNG：若是则直接等比缩放；若不是则抠图后等比缩放。
  - **联合 logo 构成**：前面一个 logo（可替换）— 中间 22×22 白色 X 按钮（无背景，线条 **#FFFFFF**，居中对齐，前后各 20px，上下左右对称）— 后面固定 logo，整体高度 50px。
- **实现**：布局由 `banner-spec` 的 `LAYOUT_BY_CANVAS[(1196, 672)]` 提供；联合 logo 由 `scripts/combine_joint_logo.py` 生成后贴入。

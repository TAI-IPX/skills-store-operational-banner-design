# 画布尺寸预设

默认尺寸为 **1976 × 464**。实际运营中会用到约 20 种不同尺寸/比例，此处列出常用预设；未列出的尺寸可通过脚本直接传 `--width` 与 `--height` 使用。

| 预设名 (preset) | 宽 | 高 | 说明 |
|-----------------|-----|-----|------|
| default         | 1976 | 464 | 默认首屏/通栏（规范1） |
| wide            | 3320 | 500 | 宽幅通栏（规范2），顶部 40px 纯白 |
| card-500        | 500  | 280 | 卡片 500×280，渐变左→右 |
| card-304        | 304  | 216 | 田字格 304×216，安全区 x=0-303 y=0-120，渐变 y=90-216 上→下 30% 边缘柔和 |
| strip           | 1470 | 200 | 无文字条幅，仅背景与安全区 |

脚本通过 `--preset default` 或 `--width 1976 --height 464` 指定尺寸。

# Banner 规范说明（权威文档）

本规范为 **banner-spec** Skill 的权威文档。实现侧数据以 [../scripts/spec.py](../scripts/spec.py) 为准（PRESETS、SAFE_ZONE_BY_CANVAS、LEGEND_ZONE_BY_CANVAS、LAYOUT_BY_CANVAS、GENRE_PRESETS、OUTPUT_FILENAME_BY_PRESET）。与 banner 相关的其他 Skill（裁切、扩图、背景生成、合成等）应引用本规范并从此 Skill 读取数据。

**文字与坐标约定**：所有主标题、副标题、按钮等文字的位置与字号均以本规范及 `spec.py` 中 `LAYOUT_BY_CANVAS` 的**坐标像素与字号**为准。新增或修改某尺寸规范时，须在本文档与 `spec.py` 的 `LAYOUT_BY_CANVAS` 等中同时更新。

---

## 1. 画布与命名

- **默认尺寸**: 宽 1976 × 高 464（像素）。
- **多尺寸**: 具体列表与命名见 [presets.md](presets.md)。脚本支持通过预设名或直接传宽高指定画布。
- **各尺寸命名为**：1976×464 → 首页 1976×464；3320×500 → **专题长图 3320\*460**（输出文件名 `专题长图 3320x460.png`，`*` 在文件名中写作 `x`）；500×280 → 专题封面 500\*280；304×216 → **商店田字格 304\*216**（输出文件名 `商店田字格 304x216.png`）；1740×220 → 专题头图 1740×220；984×442 → 商店移动端banner 984\*442；650×275 → 商店移动端专题封面 650\*275；720×220 → 商店移动端专题头图 720\*220。

## 5. 安全区（共享规范）

- **安全区按画布尺寸规定为固定数值，不做比例换算**；更换画布尺寸后使用该尺寸对应的另一组固定值。数据见 `spec.py` 中 `SAFE_ZONE_BY_CANVAS`。
- **画布 W=1976、H=464**（规范1）安全区（固定）：**x = 752～1457**，**y = 0～464**。
- **画布 W=3320、H=500**（商店专题长图 3320\*460，命名见上）安全区（固定）：**x = 1470～2464**，**y = 0～500**。顶部 **y = 0～40**：该区域为**纯白色背景**填充；在 **x = 1470～2464**、**y = 40～500** 区域内根据画面内容在纯白背景上可有画面内容填充（由背景流程与合成规范共同保证）。
- **画布 W=500、H=280**（专题封面）安全区：x=200～500，y=50～280。
- **画布 W=304、H=216**（商店田字格 304×216）安全区（固定）：x=0～303，y=0～140。
- **画布 W=1740、H=220**（专题头图）：安全区 x=510～1312，y=76～172；只做画面裁切对齐，无文字填充。
- **画布 W=984、H=442**（商店移动端banner 984×442）安全区（固定）：**x = 480～964**，**y = 30～382**。
- **画布 W=650、H=275**（商店移动端专题封面 650×275）安全区（固定）：**x = 300～630**，**y = 30～215**。
- **画布 W=720、H=220**（商店移动端专题头图 720×220）安全区（固定）：**x = 330～700**，**y = 30～160**。
- **用途**：banner-composer 主/副标题须落在此区域内；裁切、扩图、背景生成等应将核心画面/主体控制在安全区内。

## 5.7 对话框区域（dialog zone）

- **对话框区域按画布尺寸规定为固定数值**，数据见 `spec.py` 中 `DIALOG_ZONE_BY_CANVAS`。
- 对话框作为后续叠加层（文字容器），由 PIL 程序化绘制六边形横幅，颜色自动从背景图 `dialog_rect` 区域取色。
- **画布 1920×550**（商店畅玩卡）对话框区域：**x=516–1021, y=363–410**（505×47px）。`compose_banner` 通过 `--dialog` 参数接收 RGBA 透明 PNG 粘贴；`_paste_dialog()` 保留 alpha 通道，六边形外区域不遮挡背景。

## 5.8 文字艺术字区域（text art zone）

- **文字艺术字区域按画布尺寸规定为固定数值**，数据见 `spec.py` 中 `TEXT_ART_ZONE_BY_CANVAS`。
- 文字艺术字走独立管线：API 生图 → 亮度蒙版抠图 → 粘贴。抠图不使用 BiRefNet，改用 `gray.point(lambda x: 255-x)` 或 `x`（根据底色深浅自动判断）。
- **画布 1920×550**（商店畅玩卡）文字艺术字区域：**x=516–1026, y=90–360**（510×270px）。`compose_banner` 通过 `--text-art` 参数接收 RGBA 透明 PNG 粘贴到 `text_art_rect`。

## 5.9 排除区（exclusion zones）

- **排除区按画布尺寸规定为固定数值列表**，数据见 `spec.py` 中 `EXCLUSION_ZONES_BY_CANVAS`。
- 画面主要内容需避开这些区域（后续叠对话框/文字艺术字等组件），在文生图 prompt 中作为避让约束。
- **画布 1920×550** 排除区：文字艺术字区域 (516,1026,90,360) + 对话框区域 (516,1021,363,410)。

## 6. 输出

- **输出目录**: 默认 `./output/`。
- **格式**: 默认 PNG；支持 JPG。
- **按规范约定的输出文件名**（`spec.py` 中 `OUTPUT_FILENAME_BY_PRESET`；Windows 下用 x 代替 *）：
  - 1976×464 → `首页 1976x464.png`
  - 3320×500 → `专题长图 3320x460.png`
  - 500×280 → `专题封面 500x280.png`
  - 304×216 → `商店田字格 304x216.png`
  - 1740×220 → `专题头图 1740x220.png`
  - 984×442 → `商店移动端banner 984x442.png`
  - 650×275 → `商店移动端专题封面 650x275.png`
  - 720×220 → `商店移动端专题头图 720x220.png`
  - Legend zone 系列见 presets.md 或 `spec.py` 中 `OUTPUT_FILENAME_BY_PRESET`。

## 7. 字体

- **唯一合规字体**: 微软雅黑（主标题：Bold，副标题：Regular）。安装说明见 banner-composer 的 [install_font.md](../../banner-composer/references/install_font.md)（若存在）。

## 8. 组件规范

- **气泡提示框**：见 [bubble_dev.md](./bubble_dev.md)，支持 4 种类型：
  - 商店弹泡新规（有关闭按钮）
  - 商店弹泡旧规（无关闭按钮）
  - 浏览器弹泡旧规（圆角矩形 + 左侧箭头）
  - 浏览器弹泡新规（圆角矩形 + 底部尾巴，默认合并导出）
- **联合 Logo**：见 [joint_logo.md](./joint_logo.md)，图1 — 20px — 22×22 X 按钮 — 20px — 图2，整体高度 50px，输出透明底 PNG。默认素材为 `assets/logo1.png` + `assets/logo2.png`。

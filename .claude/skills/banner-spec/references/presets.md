### Step 1 风格自动注入

使用 `-g 开放平台` 跑 `run_full_with_custom_prompt.py` 时，Step 1（文生图）会自动从 `spec.py` 的 `GENRE_STYLE_PROMPT["开放平台"]` 读取风格描述片段，拼到用户 description 之后，确保即梦/Gemini 生成的背景图符合开放平台的构图与风格约束（左侧留白、右侧单一主视觉、蓝色渐变、禁止芯片/多设备示意图等）。无需每次手动在描述里重复写风格。

### Banner 颜色参考（即梦/Gemini 生成背景时遵循）

根据产品品牌配色，建议：

- **主色 + 同色系辅色**：主色（如皇家蓝/钴蓝）搭配同色系辅色，例如：鲜绿/天蓝/浅蓝/淡紫/浅薄荷绿/浅灰蓝/浅青等，用于背景渐变与大面积色块。
- **主色 + 对比色**：主色搭配对比色（如黄、橙、深紫等）时，**对比色仅建议小面积使用**，如元素点缀、图标、高光，不占主视觉。

### 风格与素材参考（与已产出 Banner 保持一致）

- **主色与渐变**：蓝色系为主（深蓝、皇家蓝、天蓝、电光蓝）；背景多为平滑蓝渐变（左上→右下或左→右），可带柔和光影/光带。
- **辅色**：浅紫、品红/粉、金、橙黄、薄荷绿等小面积点缀；主标题/副标题用白色，按钮白字+浅底或白描边。
- **氛围**：科技感、现代、清爽、冷色；可含半透明 3D、磨砂玻璃感、流线型光带；信息清晰、留白充足，主体偏右或居中偏右，左文右图。
- **即梦/Gemini 提示词方向**：蓝色渐变背景、科技感、现代简洁、半透明 3D 元素、柔和光影、冷色调、留白充足、主色为蓝辅以少量紫/金/橙点缀；不在图内生成文字。

### 主元素风格参考（设计元素使用建议）

根据业务熟悉度与场景，主元素可从以下两个维度选择并组合：

**设计风格（三选一或组合）：**
- **毛玻璃**：半透明叠层、磨砂质感；前层浅蓝毛玻璃 + 后层深蓝块，可带简洁图标。提示词可含：frosted glass, glassmorphism, translucent layers, soft blur。
- **3D**：立体层/卡片叠放、有厚度与透视；可带暖色小块点缀，偏数据/焦点感。提示词可含：3D layers, stacked cards, depth, subtle warm accent。
- **简洁扁平化 / 2.5D**：扁平图形 + 轻微等距立体感（如云 + 2.5D 底座），简洁带层次。提示词可含：flat design, 2.5D isometric, simple shapes, soft depth。

**元素主体（三选一或组合）：**
- **简洁几何图形**：三角、圆、方等基础几何，色块明确（蓝/绿/红等）。提示词可含：simple geometric shapes, triangle circle square, minimal。
- **科技化图形线条**：线框立方体、网格、光标/箭头等，偏数字、科技感。提示词可含：wireframe, grid, tech lines, holographic, digital。
- **具象业务图形**：抽象化业务图标（如 2×2 格 + 叠层方块），现代简洁。提示词可含：abstract business icon, grid with overlay, modern UI icon。

生成主元素时：先选一种设计风格 + 一种元素主体，再结合上文配色与氛围一并写入 prompt。
